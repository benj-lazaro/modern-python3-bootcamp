Ints and Floats
---------------
- Python has two main types of numbers
    - int (integers)
        - Whole numbers
        - e.g. 1, 4, -10

    - float (floating point)
        - Decimal numbers
        - e.g. 6.01, 1.333, 0.0

    - NOTE: Calculating an int with a float
        - ALWAYS evalutes to float


- The function type() identify the data type of a value
