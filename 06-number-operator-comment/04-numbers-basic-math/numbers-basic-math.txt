Numbers: Basic Math
-------------------
- Commonly used Math Operators
    - Addition              +
    - Subtraction           -
    - Multiplication        *
    - Division              /
    - Division (integer)    //
    - Exponentiation        **
    - Modulo                $

    - Python's Division operation
        - It ALWAYS returns a Float


- Python's mathematical order of operations
    - PEMDAS (highest to lowest)
        - Parenthesis
        - Exponentiation
        - Multiplication
        - Division
        - Addition
        - Subtraction

    - To override the order
        - Enclose the operands and operators within a pair of ( )
